                          Code Bridge
### Overview
Code Bridge is an AI-powered learning platform designed to help students, especially in Ethiopia, overcome challenges in learning programming caused by language barriers.

Many students understand logic but struggle with English-based explanations, complex syntax, and confusing error messages. Code Bridge bridges this gap by acting like a friendly AI tutor, delivering explanations in a simple and student-friendly way that matches how learners naturally think.
### Key Features
* AI Assistant – Chat-based coding help with explanations, examples, and debugging support
* Structured Courses – Learn Python, JavaScript, Java, SQL, and more
* Beginner-Friendly Notes – Simple, clear explanations inspired by platforms like W3Schools
* Language Support – Toggle between English and Amharic for better understanding
* Offline Access – Download courses and notes for learning without internet
* Responsive Design – Works smoothly on desktop and mobile devices

### Tech Stack
Frontend: React (Vite), Tailwind CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
AI Integration: OpenAI API
### Project Structure
The project follows a clean and scalable full-stack architecture with separate frontend and backend systems, making it easy to maintain and extend.
### Vision
To make programming education accessible, understandable, and engaging for every student regardless of language barriers.
### Why Code Bridge
Because learning to code should not feel confusing or frustrating. It should feel like having a smart friend who explains everything clearly.
